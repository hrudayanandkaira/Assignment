
# Data lake project using Terraform

## RESOURCESS
- AWS SNS
- AWS EMR
- AWSDECURITY GROUP
- AWS LAKEFORMATION
- AWS IAM
- AWS GLUE
## Usage

To run this example you need to execute:

```bash
$ terraform init
$ terraform plan
```
